
# ✊✋✌️ Rock Paper Scissors using AI/ML

## 📌 Overview
This project implements a Rock Paper Scissors game using Artificial Intelligence and Machine Learning concepts. The model learns patterns from user moves and predicts the next move to play against the user intelligently.

## 🎯 Objective
To develop an AI-based game that can predict player behavior and improve decision-making using machine learning techniques.

## 🧠 Approach
- Data collection from user gameplay
- Pattern recognition in moves
- Feature extraction from previous actions
- Training a simple predictive model
- Real-time prediction against the user

## 🛠️ Technologies Used
- Python  
- NumPy / Pandas  
- Scikit-learn
- Random module (baseline logic)  

## 🤖 AI Concept Used
- Supervised learning / sequence pattern prediction  
- Probability-based decision making  

## 🎮 Game Logic
- User selects Rock, Paper, or Scissors  
- AI predicts next move based on previous patterns  
- Winner is decided based on standard game rules  

## 📊 Outcome
The AI improves its predictions over time by learning from user behavior, making the game more intelligent and interactive.

## 🚀 Purpose
To demonstrate how machine learning can be applied in simple interactive games and real-time prediction systems.
